Install wiringPi dependencies before compiling the kernel module

Pins 17, 22,27 and 5 are configured as output pins (input for the 7447 decoder IC)
Pin 7 is to be connected to the switch to be grounded
Pins 1 and 6 are connected to vcc and ground respectively

setup.sh file need to given permission to be executed 
 
run: ./setup.sh
